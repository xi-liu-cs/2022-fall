definition 3.1.1, $S \subset \mathbb{R}, x \in \mathbb{R}$\\
cluster point: $\forall \varepsilon > 0, (x - \varepsilon, x + \varepsilon) \cap S \setminus \{x\}$ is not empty\\
not cluster point: $\exists \varepsilon > 0, (x - \varepsilon, x + \varepsilon) \cap S \setminus \{x\}$ is empty\\
cluster point: $\forall \varepsilon > 0, \exists y \in S, y \not= x, |x - y| < \varepsilon$\\
not cluster point: $\exists \varepsilon > 0, \forall y \in S, y \not= x, |x - y| \geq \varepsilon$\\
proposition 3.1.2, $S \subset \mathbb{R}, x \in \mathbb{R}$ is a cluster point of $S$ if and only if there exists a convergent sequence of numbers $\{x_n\}$ such that $\forall n \in \mathbb{N}, x_n \not= x, x_n \in S, \lim x_n = x$\\
\\
(a)\\
$\forall a, b \in \mathbb{R}, a < b, S = (a, b)\\
\forall x \in [a, b], \forall \varepsilon > 0$,\\
if $x, y \in \mathbb{R}, x < y, \text{ then } \exists r \in \mathbb{Q}, x < r < y$ \qquad (theorem 1.2.4.ii)\\
since $x - 1 / n, x + 1 / n \in \mathbb{R}, \text{ so } \exists x_n \in \mathbb{Q}, x - 1 / n < x_n < x + 1 / n\\$
if $\{x_n\}$ converges to some $L \in \mathbb{R}$, then $\forall \varepsilon > 0, \exists M \in \mathbb{N}, \forall n \geq M, |x_n - L| < \varepsilon$,\\
let $M \geq 1 / \varepsilon\\
\text{then } \forall n \geq M, 1 / n \leq 1 / M \leq \varepsilon\\
x - \varepsilon \leq x - 1 / n \leq x_n \leq x + 1 / n \leq x + \varepsilon\\
x - \varepsilon \leq x_n \leq x + \varepsilon\\
-\varepsilon \leq x_n - x \leq \varepsilon\\
|x_n - x| \leq \varepsilon\\
\text{then }\forall \varepsilon > 0, \exists M \geq 1 / \varepsilon, \forall n \geq M, |x_n - x| < \varepsilon$\\
so $\lim_{n \rightarrow \infty} x_n = x$\\
based on proposition 3.1.2\\
$x_n \in (x - \varepsilon, x + \varepsilon) \cap S \setminus \{x\}$ is not empty, so $x$ is a cluster point of $S$\\

\\
(a)\\
\begin{align*}
\forall \varepsilon &> 0\\
|f(x) - f(c)| &< \varepsilon\\
|\sqrt{x} - \sqrt{c}| &< \varepsilon\\
|\sqrt{x} - \sqrt{c}||\sqrt{x} + \sqrt{c}| &< \varepsilon |\sqrt{x} + \sqrt{c}|\\
|x - c| &< \varepsilon|\sqrt{x} + \sqrt{c}|
\end{align*}
$\exists i \in \mathbb{R}, i > 0, |x - c| < i$, then $x - c < i, c - i < x < c + i\\
\sqrt{x} < \sqrt{c + i}\\
\sqrt{x} + \sqrt{c} < \sqrt{c + i} + \sqrt{c}$\\
\begin{align*}
|x - c| &< \varepsilon|\sqrt{x} + \sqrt{c}| = \varepsilon(\sqrt{x} + \sqrt{c}) < \varepsilon(\sqrt{c + i} + \sqrt{c})\\
\exists \delta &:= \min(i, \varepsilon(\sqrt{c + i} + \sqrt{c}))\\
|x - c| &< \delta\\
\forall c \in [0, \infty), \lim_{x \rightarrow c} f(x) &= \sqrt{c}
\end{align*}
$f(x) = \sqrt{x}$ is a continuous function\\
